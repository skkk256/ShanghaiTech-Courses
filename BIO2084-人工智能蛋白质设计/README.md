
# 人工智能蛋白质设计

**学分**: 1
**授课老师**: 豆佳宜


## 课程评价

暑学期上的 PNP 课程。以了解和图一乐为主，讲的内容还是比较多的，从蛋白质的四级结构到一些生物学上的原理，以及 AI 的一些基本方法、网络结构和AlphaFold1-3都会涉及。

作业是从 PDB 里找一个自己感兴趣的蛋白，拿 ProteinMPNN 跑个序列，然后用 colab 跑下 alphafold2.

期末 Report 是二选一

    1. 学习本门课程的感想和收获。可以是对整体内容的感想和深度探索；也可以是就本课程中某一知识点展开后阅读文献进行自我探索的总结；需要附上所读参考文献。

    2.学习本门课收到启发而提议的蛋白质设计项目(小型的科研计划书)；需要附上所读参考文献和自己的想法说明。
