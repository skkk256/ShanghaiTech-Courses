## 详细信息

第一门 workload 巨大的课, 爆改CS61C, 4+2的6分大课 (110+110P), 三种类型作业, 2次期中1次期末

从现在来看课程质量确实不算差, 尤其是从 CS61C 抄来的整体结构, 先自顶向下从C到risc-v，讲高级语言的 compile, assemble, link, load; 然后再自底向上, 从数字电路讲到 CPU 实现: datapath, pipeline等等，至少逻辑非常畅通

实在苯人菜鸡+状态稀烂, 超出能力范围, 拼尽全力不能, 被lab/hw/project折磨的死去活来, 第一次期中怒拿 38/100, 第二次期中高平均14, 期末堪堪高平均2分, lab和coding hw应该辛苦拿满了, 纸质作业一坨, 最后砍下 B, 延续了从CS101开始的悲惨大学生活

详细内容见 [课程网站](https://toast-lab.sist.shanghaitech.edu.cn/courses/CS110@ShanghaiTech/Spring-2023/index.html)
