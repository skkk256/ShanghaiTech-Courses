## 详细信息

以板书为主，除部分补充材料外基本没有课件

作业为课本 Elements.of.Information.Theory 的课后习题, 有答案

- HW1: 2.1, 2.3, 2.5, 2.12, 2.14
- HW2: 2.16, 2.25, 2.46(optional), 4.33, 5.4, 5.6
- HW3: 5.12, 5.20, 5.30, 3.1, 3.9
- HW4: 7.2, 7.3, 7.4, 7.13, 7.34, 7.37
- HW5: 8.1, 8.3, 8.8, 8.9
- HW6: 9.1, 9.2, 9.5, 9.6, 9.9, 9.15 

### Project

复现给定或自选论文，不强制要求有创新。24Fall以poster形式做展示，之前为上课Pre

### 期末考试

给分以考试为主，占比50%，6道大题，前4题比较基础、后2题较难
