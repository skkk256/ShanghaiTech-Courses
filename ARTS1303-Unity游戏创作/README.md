## 课程评价

一门有趣但是 workload 较大的课

Unity 入门课，Brian Cox 神中神

有谁会不喜欢做游戏呢？


## 详细信息

极少数英文上课但是基本能听懂的课了，在创艺机房上课，一遍演示一遍做

3次作业，分别是对应进度的技术要求和游戏类型要求（苯人第二次没看到要求平台跳跃x），前车之鉴是有点过度使用 unity asset store 的资产和插件，导致项目过于庞大、繁琐。但是如果要卷出好的效果，多找些美术、动作资产还是很必要的。

Final Project，三到四人组队，是普遍能做到比较卷的程度，被大佬带飞，做了版复刻雨中冒险。

因为 workload 比较大（主要是大伙太卷了，而且做游戏本身还是有很多dirty work），非常不建议在暑学期选课
