
# 计算机图形学I

**学分**: 4
**授课老师**: 刘晓培


## 课程评价

一门 workload 比较大但是也比较有收获的课

CG 是你校本科生专业课公认比较硬核的课（虽然23FALL似乎宽松了一些）。最大的难点可能是因为这是本科生碰到的第一门理论（数字、physical base）和工程结合比较紧密的课程；而且考核内容也比较多，有 quiz、pa、project还有考试，opengl 也不怎么好 debug

内容重心放在 Ray Tracing 上，前期先照例会讲相机模型、光栅化，然后到渲染方程 BRDF 等等，还会讲一些其他几何、模拟、体渲染之类，会稍微比 games101 多一些

## 详细信息

quiz 不定期会有，主要是公式推导和计算，苯人反正不会，没几次写出来过

会有五次编程作业，opengl入门，贝塞尔/B样条，BRDF，体渲染，布料模拟；每次还会有若干额外bonus可选；第三次工程量和难度显著大些（考虑到大小，仓库中的 hw 把 glfw, eigen 等 lib 全删了）

Project 选项很多，分 ABCD 多种难度，还有不同人数的可选，都比较有意思；苯人选了碰撞检测，几乎是从 reference 的 GamePhysicsOneWeek 一通抄

Paper Reading 就是 Paper Reading

期末考试就是期末考试，23FALL不是很难感觉
