
# 三维视觉概念设计

**学分**: 2
**授课老师**: 高鸿晟


## 课程评价

Blender 入门课，非常好课。高老师讲 blender 讲得很好，而且人非常 nice。

主要课程内容是渲染，对建模和雕刻要求不高，上课也会讲一些偏艺术和设计类的，比方说如何打光，如何构图，相机参数选择等等

第一次作业为设计和搭建一个纪念碑谷场景

会有一次课堂练习是在一个给定的场景中，自己选择合适角度和镜头出一组图

其余为期末Project以及定期的汇报，主要是自己设计一个故事背景，搭建场景，然后渲染组图，可以尽可能地用网上的资源（focus 设计、渲染），老师也会提很多建议；本人选了一个复刻《来自深渊》的场景，结果花了很大地力气去hack和解包官方游戏，但被UE的加密和搞不明白的 uassets 干烂了，最后还是拿淘宝素材糊了糊
