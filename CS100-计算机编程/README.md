## 课程评价

c + cpp. 非常不错的入门课, 八成质量靠 知名TA GKXX

[习题课 slides](https://github.com/GKxxUCAS/CS100-recitations-spring2024)

作业花点时间基本上能做出来, 但也因此导致作业上很难有区分度, 最后全看期中考试了

不过作为编程入门课, 确实多看多写还是最重要的, 非常亮点的是最后两次作业照例是完成一个游戏 ( 22 SPRING 是飞机大战, 往后是植物大战僵尸 ), 给的代码框架很完善, 对于像 OOP/容器/coding技巧 这类的理解和训练也很有帮助!
