
# 深度学习

**学分**: 4
**授课老师**: 许岚


## 课程评价

研究生必修课所以上的人很多，Workload 比较小，老师助教摆摆的，
内容上的还行其实，比较全面，后期主要讲生成模型，像VAE、GAN、diffusion的推导，该有的都有。


### 作业

3次万年不变的照Stanford cs231n改的编程作业（理论上，24FALL只发了2次）, 第一次是KNN, SVM, MLP; 第二次是 GAN

抽若干节课会有当堂 quiz，感觉上有点难度，例如会有 diffusion 的推导，但是大伙都有 GPT，所以……

### Project

自选课题，内容普遍会非常多样，从蛋白质到影像到通信，因为主要是研究生课，所以基本会拿自己课题出来凑活凑活
