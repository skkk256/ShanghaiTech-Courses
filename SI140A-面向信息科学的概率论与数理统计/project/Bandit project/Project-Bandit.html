def epsilon_greedy(n, epsilon):
    # initialize
    theta = np.array([0.5, 0.7, 0.4]) #the true distribution
    theta_hat = np.zeros(3)
    arm_count = np.zeros(3)
    rewards = np.zeros(n)
    
    for t in range(n):
        # choose an arm
        if np.random.random() < epsilon:
            # explore
            arm = np.random.choice(3)
        else:
            # exploit
            arm = np.argmax(theta_hat)
        
        # draw a reward from the chosen arm
        reward = np.random.binomial(1, theta[arm])
        
        # update the estimate of the chosen arm
        arm_count[arm] += 1
        theta_hat[arm] += (reward - theta_hat[arm]) / arm_count[arm]
        
        # record the reward
        rewards[t] = reward
        
    return np.sum(rewards)

def UCB(n, c):
    theta = np.array([0.7, 0.5, 0.4]) #the true distribution
    arm_count = np.zeros(3)
    theta_hat = np.zeros(3)
    rewards = np.zeros(n)

    for t in range(3):
        #select the arm
        arm = t
        arm_count[arm] = 1
        reward = np.random.binomial(1, theta[arm])
        theta_hat[arm] = reward
        rewards[t] = reward
        
    for t in range(3, n):
        #select the arm
        theta_hat_tmp = theta_hat + c * np.array(
            [np.sqrt(2 * np.log(t+1) / arm_count[i]) for i in range(0,3)])
        arm = np.argmax(theta_hat_tmp)
        
        # draw a reward from the chosen arm
        reward = np.random.binomial(1, theta[arm])
        
        # update the estimate of the chosen arm
        arm_count[arm] += 1
        theta_hat[arm] += (reward - theta_hat[arm]) / arm_count[arm]

        # record the reward
        rewards[t] = reward
        
    return np.sum(rewards)

def TS(n, a , b):
    alpha = np.array(a)
    beta = np.array(b)
    
    theta_hat = np.zeros(3)
    theta = np.array([0.7, 0.5, 0.4]) #the true distribution
    rewards = np.zeros(n)

    for t in range(n):
        #sample the model
        for j in range(3):
            theta_hat[j] = np.random.beta(alpha[j], beta[j])
        #select the arm
        arm = np.argmax(theta_hat)
        # draw a reward from the chosen arm
        reward = np.random.binomial(1, theta[arm])
        rewards[t] = reward
        alpha[arm] += reward
        beta[arm] += 1 - reward
        
    return np.sum(rewards)

  1. Give your understanding of the exploration-exploitation trade-off in bandit algorithms.

Your anwser of problem 5 in Part I

  • Greedy algorithm is a direct thinking to use exploration-exploitation, we have probability $\epsilon$ to explore which is represented as randomly selecting an arm, and the probability $1-\epsilon$ to exploitation which means we choose $\underset{j \in\{1,2,3\}}{\arg \max }\ \widehat{\theta}(j)$
  • UCB algorithm has a exploration-exploitation trade-off when choose arms. Firstly, we will choose the third arms one by one. Next, we choose the best estimation. Each Arm has a confidence interval for the mean value of the return, which becomes narrower as the number of trials increases (gradually determining whether the return is generous or poor). Before each selection, the mean value and confidence interval of each arm are re-estimated based on the results of the previous trials. The arm with the highest confidence interval is selected.
  • TS algorithm is essentially different from the previous two algorithms. It does not use a fixed parameter to adjust the probability of exploration, but contains the trade-off between exploration and exploitation in the comparison of the expectation of each arms income. The prior distribution we present affects the final payoff in two ways. On the one hand, it is to predict the income of each arm. The more accurate this prediction is, the lower the trial and error cost in the experiment, and the optimal arm can be selected more accurately from the beginning. On the other hand, there is the number of experiments of the prior distribution, that is, the size of $\alpha$ and $\beta$. If these two values are relatively large, it means that we have done a large number of experiments in advance. If the prediction is accurate, the loss caused by the contingency of the initial several experiments can be effectively avoided.

def TS_dependent(n, a , b):
    alpha = np.array(a)
    beta = np.array(b)
    theta = np.array([0.7, 0.5, 0.4])
    theta_hat = np.zeros(3)
    rewards = 0

    for t in range(n):
        #sample the model
        for j in range(3):
            theta_hat[j] =  np.random.beta(alpha[j], beta[j])
        #select the arm
        arm = np.argmax(theta_hat)
        # draw a reward from the chosen arm
        reward = np.random.binomial(1,theta[arm])
        rewards += reward
        alpha[arm] += reward
        beta[arm] += 1 - reward
        # Success or not on one arm will also affect another arm
        for j in range(3):
            if j != arm:
                beta[j] += reward
                alpha[j] += 1 - reward
    return rewards

Part II: Bayesian Bandit Algorithms¶

There are two arms which may be pulled repeatedly in any order. Each pull may result in either a success or a failure. The sequence of successes and failures which results from pulling arm $i$ ($i \in \{1, 2\}$) forms a Bernoulli process with unknown success probability $\theta_{i}$. A success at the $t^{th}$ pull yields a reward $\gamma^{t-1}$ ($0 < \gamma <1$), while an unsuccessful pull yields a zero reward. At time zero, each $\theta_{i}$ has a Beta prior distribution with two parameters $\alpha_{i}, \beta_{i}$ and these distributions are independent for different arms. These prior distributions are updated to posterior distributions as arms are pulled. Since the class of Beta distributions is closed under Bernoulli sampling, posterior distributions are all Beta distributions. How should the arm to pull next in each time slot be chosen to maximize the total expected reward from an infinite sequence of pulls?

  1. One intuitive policy suggests that in each time slot we should pull the arm for which the current expected value of $\theta_{i}$ is the largest. This policy behaves very good in most cases. Please design simulations to check the behavior of this policy.

    return reward

\begin{equation*} \begin{aligned} R_{1}(\alpha_{1},\beta_{1}) = & \frac{\alpha_{1}}{\alpha_{1}+\beta_{1}} [1+\gamma R(\alpha_{1} + 1, \beta_{1}, \alpha_{2}, \beta_{2})] \\ & + \frac{\beta_{1}}{\alpha_{1} + \beta_{1}} [\gamma R(\alpha_{1}, \beta_{1} + 1, \alpha_{2}, \beta_{2})]; \\ R_{2}(\alpha_{2}, \beta_{2}) = & \frac{\alpha_{2}}{\alpha_{2} + \beta_{2}} [1 + \gamma R(\alpha_{1}, \beta_{1}, \alpha_{2} + 1, \beta_{2})] \\ & + \frac{\beta_{2}}{\alpha_{2} + \beta_{2}} [\gamma R(\alpha_{1}, \beta_{1}, \alpha_{2}, \beta_{2} + 1)]; \\ R(\alpha_{1}, \beta_{1}, \alpha_{2}, \beta_{2}) = & \max \left\{ R_{1}(\alpha_{1}, \beta_{1}), R_{2}(\alpha_{2}, \beta_{2}) \right\}. \end{aligned} \end{equation*}

Your anwser of problem 3 in Part II

Here we take $R_1(\alpha_1,\beta_1)$ into two parts: the first time the rewards we get if we select the first arm with the rewards afterwards.

First if the first trial is a success, then afterwards, we need to add rewards $\gamma^{(t-1)}=1$, and we turn $\alpha_1 $ to be $\alpha_1+1$, and also we need to multiply the rewards afterwards by $\gamma$.

Next if we failed on the first trial, then there is no need to add rewards, so that is: $$ \begin{align*} R_{1}(\alpha_{1},\beta_{1}) = & \frac{\alpha_{1}}{\alpha_{1}+\beta_{1}} [1+\gamma R(\alpha_{1} + 1, \beta_{1}, \alpha_{2}, \beta_{2})] \\ & + \frac{\beta_{1}}{\alpha_{1} + \beta_{1}} [\gamma R(\alpha_{1}, \beta_{1} + 1, \alpha_{2}, \beta_{2})]; \end{align*} $$

and it's the same with $R_2$, due to our algorithm, the first trial should be the arm with $\theta$ expectation bigger, so we know that is:

$$ \begin{align*} R(\alpha_{1}, \beta_{1}, \alpha_{2}, \beta_{2}) = & \max \left\{ R_{1}(\alpha_{1}, \beta_{1}), R_{2}(\alpha_{2}, \beta_{2}) \right\}. \end{align*} $$

def Bayesian_alg1(a1, b1, a2, b2, n, gamma):
    reward = 0
    success = 0
    for i in range(n):
        expect1 = a1/(a1+b1)
        expect2 = a2/(a2+b2)
        if expect1 > expect2:
            proba = np.random.beta(a1, b1)
            if (np.random.binomial(1, proba)):
                a1 += 1
                reward += gamma**i
                success += 1
            else:
                b1 += 1
        else:
            proba = np.random.beta(a2, b2)
            if (np.random.binomial(1, proba)):
                a2 += 1
                reward += gamma**i
                success += 1
            else:
                b2 += 1
    return reward
