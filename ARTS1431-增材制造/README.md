# 增材制造

**学分**: 3
**授课老师**: 张振波、顾小煜、翟梓融


## 课程评价

和我想象中略有差别的课。老师明确表示“不希望这门课只是随便找个模型，然后打印一下完事”，所以融入了非常多材料相关、设计相关的内容。

期间教了SolidWorks、rhino/grasshopper、fusion 360等一堆软件，并且要求最后项目产出能体现对3D打印流程的改进和体现出设计的要素。

个人认为内容过多过杂，比方说CS学生基本不会去听上课的一堆材料的内容，又比方说上课教了非常多CAD软件（不过都浅尝辄止），但最后普遍大家都是拿blender搓的，还比方说个人认为“体现出对3D打印技术的改进”几乎是不大现实的。应该做一些拆分，简化，主要focus在某一点上。
