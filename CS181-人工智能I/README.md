## 课程评价

不错的课，虽然是爆改Berkeley CS188。

took away 讲课很好，虽然也是少数 Chinese 但是偏要英文授课的神仙；不过因为还 Chinglish 相对容易听懂很多

作业有两种，一个是 blackboard 上发的选择和填空；另一个是用各种方法实现吃豆人，经典 CS188 作业，答案满天飞，也有比较大一部分 workload 在读懂给的代码和弄清楚接口；但是有几次 PA 指标改的很逆天，几乎优化不出来

Project 自选题，做了一个 AI 打植物大战僵尸

考试期中+期末，类似算法，不定项选择+推算法，给分基本都看考试
